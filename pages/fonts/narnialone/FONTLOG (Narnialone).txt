Basic Font Information

"Narnialone" Font is a Unicode typeface family that supports all languages
that use the Latin kids and its variants, and could be expanded to support
other scripts.


"Narnialone" Font you'll find at: https://glottastudio.com

"Narnialone" Font is created with FontCreator.


More specifically, this release supports the following Unicode ranges:

 - Basic Latin and Controls 
 - Latin 1 Supplement And Controls 
 - General Punctuation 
 - Currency Symbols 
 - Mathematical Operators 

This release contains:

 - Narnialone.ttf
 - Narnialone.otf
 - Narnialone.woff
 - SIL - OpenFont License.txt - official license text
